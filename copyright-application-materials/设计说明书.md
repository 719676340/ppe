## 第一章 软件概述

### 1.1 项目背景

广西中烟南宁卷烟厂制丝车间作为生产核心区域，安全生产是首要任务。根据安全规范要求，进入车间重点区域的作业人员必须佩戴安全帽。传统的人工巡检方式存在效率低、覆盖面小、难以及时发现问题等不足。

为提高安全生产管理水平，制丝车间安全头盔智能检测系统采用先进的计算机视觉和深度学习技术，实现安全帽佩戴情况的自动化检测和违规行为记录，为安全生产管理提供技术支撑。

### 1.2 建设目标

1. **自动化检测**：实现安全帽佩戴情况的7×24小时自动检测
2. **实时告警**：违规行为及时发现，实时推送通知
3. **数据记录**：完整记录违规事件，便于追溯和分析
4. **统计分析**：提供多维度统计数据，辅助管理决策
5. **易用性强**：界面友好，操作简单，便于推广使用

### 1.3 应用范围

本系统主要应用于：
- 制丝车间重点安全区域
- 其他需要佩戴安全帽的生产作业区域
- 建设工地等工业安全场景

## 第四章 详细设计

### 4.1 核心模块设计

#### 4.1.1 PPE检测器模块（PPEDetector）

**功能描述**：负责对视频帧进行PPE检测，识别人员是否佩戴安全帽。

**主要方法**：

```python
class PPEDetector:
    def __init__(self, model_path: Path)
        # 初始化YOLO模型

    def detect(self, image: np.ndarray) -> List[Dict]
        # 检测图像中的PPE佩戴情况
        # 返回检测结果列表

    def set_confidence_threshold(self, threshold: float)
        # 设置置信度阈值

    def set_iou_threshold(self, threshold: float)
        # 设置IOU阈值

    def get_center_point(self, bbox: List[float]) -> Tuple[float, float]
        # 获取边界框中心点
```

**检测流程**：

```
输入图像 → YOLO模型推理 → 后处理 → 返回检测结果
           ↓
    目标检测+分类
    no_ppe/with_ppe
```

#### 4.1.2 视频流处理器模块（StreamProcessor）

**功能描述**：负责视频流的采集、处理和检测结果管理。

**主要方法**：

```python
class StreamProcessor:
    def __init__(self, source_url, camera_id, detector, roi_manager,
                 violation_recorder, on_detection_callback)
        # 初始化处理器

    def start()
        # 启动视频流处理

    def stop()
        # 停止视频流处理

    def get_latest_frame()
        # 获取最新处理后的帧

    def get_latest_detections()
        # 获取最新检测结果

    def get_status()
        # 获取处理器状态
```

**处理流程**：

```
视频流 → 读取帧 → PPE检测 → 区域判断 → 违规处理 → 结果推送
              ↓         ↓          ↓
          YOLO模型   ROI判断   记录+通知
```

#### 4.1.3 ROI管理器模块（ROIManager）

**功能描述**：管理检测区域，判断目标是否在区域内。

**主要方法**：

```python
class ROIManager:
    @staticmethod
    def parse_coordinates(coordinates_json: str)
        # 解析ROI坐标JSON字符串

    @staticmethod
    def scale_coordinates(normalized_coords, width, height)
        # 坐标归一化转换

    @staticmethod
    def point_in_polygon(point, polygon)
        # 判断点是否在多边形内（射线法）

    def is_bbox_in_zone(bbox, zone_coords)
        # 判断边界框是否在检测区域内

    def draw_zone(image, zone_coords, color, thickness)
        # 在图像上绘制检测区域
```

**区域判断算法**：

采用射线法判断点是否在多边形内：
1. 从待判断点向右发射射线
2. 统计射线与多边形边的交点数
3. 交点数为奇数则在多边形内，偶数则在外

#### 4.1.4 违规记录器模块（ViolationRecorder）

**功能描述**：处理违规检测和记录，包括去重、截图保存、数据库存储。

**主要方法**：

```python
class ViolationRecorder:
    def __init__(self, upload_dir, dedup_interval, max_distance)
        # 初始化违规记录器

    def record_violation(camera_id, zone_id, frame, detection)
        # 记录违规（含去重检查）

    def _is_duplicate(camera_id, zone_id, center_x, center_y)
        # 检查是否重复

    def _add_to_cache(camera_id, zone_id, x, y)
        # 添加到去重缓存

    def _save_violation_image(camera_id, zone_id, frame, bbox)
        # 保存违规截图

    def clear_cache(camera_id)
        # 清空去重缓存
```

**去重算法**：

基于时间和位置的双重去重：
1. 计算当前检测点与缓存中历史点的欧氏距离
2. 如果距离小于阈值（默认100像素）且时间在去重窗口内（默认30秒），判定为重复
3. 定期清理过期的缓存记录
