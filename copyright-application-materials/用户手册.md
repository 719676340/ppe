## 第一章 软件简介

### 1.1 软件概述

制丝车间安全头盔智能检测系统是专为广西中烟南宁卷烟厂制丝车间设计的智能安全检测系统。系统采用先进的深度学习技术，通过摄像头实时监控车间重点安全区域，自动检测工人是否按规定佩戴安全帽，对违规行为进行记录和报警，帮助企业管理者提高安全生产管理水平。

### 1.2 应用场景

- 制丝车间重点安全区域监控
- 生产现场安全帽佩戴检查
- 安全生产违规行为记录
- 安全管理数据分析

### 1.3 系统特点

- **非接触式检测**：无需人工干预，自动完成检测
- **实时报警**：违规行为及时发现，实时推送通知
- **智能去重**：避免重复记录，提高数据质量
- **数据统计**：多维度统计分析，辅助管理决策
- **易于部署**：支持多种摄像头接入，部署灵活

## 第二章 功能概述

### 2.1 系统功能架构

本系统主要由以下功能模块组成：

| 功能模块 | 功能描述 |
|---------|---------|
| 摄像头管理 | 管理摄像头信息，配置视频源 |

本系统支持以下用户角色：

- **系统管理员**：拥有所有功能权限
- **安全管理人员**：查看监控、违规记录和统计数据
- **普通用户**：仅查看统计数据

## 第三章 安装部署

### 3.1 系统要求

#### 3.1.1 硬件要求

| 组件 | 最低配置 | 推荐配置 |
|------|---------|---------|
| CPU | Intel Core i5 | Intel Core i7及以上 |
| 内存 | 8GB | 16GB及以上 |
| 硬盘 | 100GB可用空间 | SSD 256GB及以上 |
| 网络 | 100Mbps | 1Gbps |

#### 3.1.2 软件要求

- 操作系统：Windows 10/11、Linux (Ubuntu 20.04+)
- Python 3.9或更高版本
- Node.js 16或更高版本
- 浏览器：Chrome 90+、Edge 90+、Firefox 88+

### 3.2 安装步骤

#### 3.2.1 后端安装

1. 解压安装包，进入backend目录
2. 创建虚拟环境并激活：
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   venv\Scripts\activate     # Windows
   ```
3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
4. 配置环境变量（编辑.env文件）
5. 启动后端服务：
   ```bash
   python -m uvicorn main:app --host 0.0.0.0 --port 8000
   ```

#### 3.2.2 前端安装

1. 进入frontend目录
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动开发服务器：
   ```bash
   npm run dev
   ```
4. 访问 http://localhost:5173

### 3.3 摄像头接入

系统支持以下类型的视频源：

- RTSP视频流
- HTTP视频流
- 本地视频文件
- USB摄像头

接入步骤：
1. 登录系统
2. 进入"摄像头管理"页面
3. 点击"添加摄像头"按钮
4. 填写摄像头信息，包括名称和视频源地址
5. 保存并启动检测

## 第四章 主要功能说明

### 4.1 摄像头管理

#### 4.1.1 添加摄像头

1. 点击左侧菜单"摄像头管理"
2. 点击"添加摄像头"按钮
3. 填写摄像头信息：
   - 摄像头名称：如"制丝车间1号机"
   - 视频源地址：如"rtsp://192.168.1.100:554/stream"
   - 是否启用：勾选表示自动启动检测
4. 点击"确定"保存

#### 4.1.2 编辑摄像头

1. 在摄像头列表中找到要编辑的摄像头
2. 点击操作列的"编辑"按钮
3. 修改摄像头信息
4. 点击"确定"保存

#### 4.1.3 删除摄像头

1. 在摄像头列表中找到要删除的摄像头
2. 点击操作列的"删除"按钮
3. 确认删除操作

#### 4.1.4 启动/停止检测

1. 在摄像头列表中找到要操作的摄像头
2. 点击"启动检测"或"停止检测"按钮
3. 系统会自动开始或停止该摄像头的视频流处理

### 4.2 区域管理

#### 4.2.1 添加检测区域

1. 点击左侧菜单"区域管理"
2. 选择要添加区域的摄像头
3. 点击"添加区域"按钮
4. 在视频画面上点击绘制检测区域（至少3个点）
5. 填写区域名称，如"1号机工作区"
6. 点击"确定"保存

#### 4.2.2 编辑检测区域

1. 在区域列表中找到要编辑的区域
2. 点击操作列的"编辑"按钮
3. 重新绘制检测区域或修改区域名称
4. 点击"确定"保存

#### 4.2.3 启用/禁用区域

1. 在区域列表中找到要操作的区域
2. 点击"启用"或"禁用"开关
3. 禁用的区域将不进行检测

### 4.3 实时监控

#### 4.3.1 查看实时监控

1. 点击左侧菜单"实时监控"
2. 选择要查看的摄像头
3. 系统显示实时视频流和检测结果：
   - 绿色框：已佩戴安全帽
   - 红色框：未佩戴安全帽（违规）
   - 黄色多边形：检测区域

#### 4.3.2 检测状态显示

实时监控页面显示以下信息：

- 检测人数：当前检测到的总人数
- 违规人数：未佩戴安全帽的人数
- 检测时间：最后更新时间

### 4.4 违规管理

#### 4.4.1 查看违规记录

1. 点击左侧菜单"违规管理"
2. 系统显示所有违规记录列表
3. 每条记录包含：
   - 违规时间
   - 摄像头名称
   - 区域名称
   - 违规截图
   - 处理状态

#### 4.4.2 搜索和筛选

可以通过以下条件筛选违规记录：

- 时间范围：选择开始时间和结束时间
- 摄像头：选择指定摄像头
- 处理状态：已处理/未处理
- 区域：选择指定区域

#### 4.4.3 查看违规详情

1. 点击违规记录的"查看"按钮
2. 系统显示违规详细信息，包括高清截图
3. 可以添加备注信息

#### 4.4.4 标记已处理

1. 选中要处理的违规记录
2. 点击"标记已处理"按钮
3. 系统更新记录状态

#### 4.4.5 导出数据

1. 设置筛选条件
2. 点击"导出Excel"按钮
3. 系统生成包含截图的Excel文件
4. 下载保存到本地

### 4.5 统计分析

#### 4.5.1 区域统计

1. 点击左侧菜单"统计分析" → "区域统计"
2. 选择时间范围
3. 系统显示各区域的违规数量统计

#### 4.5.2 时段统计

1. 点击左侧菜单"统计分析" → "时段统计"
2. 选择时间范围
3. 系统显示早班、中班、夜班的违规数量对比

#### 4.5.3 趋势分析

1. 点击左侧菜单"统计分析" → "趋势分析"
2. 选择时间范围和统计粒度（日/周/月）
3. 系统显示违规数量趋势图表

#### 4.5.4 摄像头统计

1. 点击左侧菜单"统计分析" → "摄像头统计"
2. 选择时间范围
3. 系统显示各摄像头的违规数量对比

### 4.6 通知中心

#### 4.6.1 查看通知

1. 点击顶部通知图标
2. 系统显示未读通知列表
3. 点击通知查看详情

#### 4.6.2 标记已读

1. 在通知列表中选中通知
2. 点击"全部已读"按钮
3. 或点击单条通知标记为已读

## 第六章 注意事项

### 6.1 使用注意事项

1. **摄像头位置**：确保摄像头能够清晰覆盖检测区域
2. **光照条件**：保持充足且稳定的光照条件
3. **网络带宽**：确保网络带宽充足，避免视频卡顿
4. **定期检查**：定期检查系统运行状态和检测效果

### 6.2 检测效果优化建议

1. **区域绘制**：检测区域应尽量贴近实际工作区域
2. **角度调整**：摄像头角度应垂直或略微俯视
3. **背景简洁**：检测区域内应避免复杂背景
4. **距离控制**：摄像头与检测区域的距离不宜过远

### 6.3 数据备份建议

1. 定期备份违规记录数据库
2. 定期备份违规截图文件
3. 建议至少保留6个月的历史数据

### 6.4 系统安全建议

1. 定期更改系统密码
2. 限制系统访问权限
3. 及时更新系统补丁
4. 定期检查系统日志

## 第七章 常见问题

### Q1：检测不准确怎么办？

A：可能原因及解决方法：
- 检查摄像头是否清晰覆盖检测区域
- 调整检测区域范围
- 检查光照条件是否良好
- 联系技术支持进行模型优化

### Q2：视频流无法连接怎么办？

A：请检查：
- 摄像头网络连接是否正常
- 视频源地址是否正确
- 防火墙是否允许访问
- 摄像头是否支持该视频流协议

### Q3：系统提示内存不足怎么办？

A：解决方法：
- 减少同时检测的摄像头数量
- 降低视频分辨率
- 增加服务器内存

### Q4：如何导出历史数据？

A：
1. 进入违规管理页面
2. 设置时间范围筛选条件
3. 点击"导出Excel"按钮
4. 等待系统生成文件并下载
